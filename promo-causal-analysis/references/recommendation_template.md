# Recommendation Template

## Overall Strategy

Use this pattern:

"After adjusting for cyclical demand, the promotion program appears to deliver [positive / mixed / weak] incremental value. The main operational lever is [exposure / discount / mixed execution], and the recommended posture is [scale / optimize selectively / hold / reduce disturbance]."

## Full-Category Strategy

Use this pattern:

"For the full category set, prioritize categories with positive LocalGap, supportive GPS response in the supported region, and stable uplift ranks. Keep categories with weak baseline coverage or unstable uplift in the observe queue rather than scaling immediately."

## Head-Category Strategy

Use this pattern:

"For head categories, separate actions into four buckets: scale, optimize, hold, and do not disturb. Use LocalGap to quantify current contribution, GPS to judge whether more intensity is justified, and uplift to choose the order of resource allocation."

## Writing Rule

Recommendations must state both:

- the evidence layer supporting the action
- the main caveat limiting overconfidence
